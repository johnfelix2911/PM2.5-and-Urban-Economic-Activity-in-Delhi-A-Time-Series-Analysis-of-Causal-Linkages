# PM2.5 and Urban Economic Activity in Delhi: A Time Series Analysis of Causal Linkages

## Project Overview
This project aims to analyze the causal linkages between PM2.5 pollution levels and urban economic activities in Delhi, using time series analysis techniques. The goal is to understand how changes in air quality impact economic performance, and vice versa.

## Datasets Used
- **PM2.5 Data**: This dataset includes hourly PM2.5 concentrations in various locations in Delhi.
- **Economic Indicators**: Data collected from various sources reflecting urban economic activities, such as GDP, employment rates, and industrial output.

## Analysis Structure
- **Data Preprocessing**: Cleaning and preparing the datasets for analysis, including handling missing values and normalizing data ranges.
- **Time Series Analysis**: Implementing techniques such as ARIMA, VAR, and Granger causality tests to explore the relationships between the datasets.
- **Visualization**: Creating visual representations of the data to illustrate trends and causal relationships.

## Conclusion
The findings from this analysis showcase the interdependencies between air quality and economic activity in Delhi. Understanding these linkages can inform policy decisions and public health initiatives.
